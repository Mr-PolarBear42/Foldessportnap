# 🏆 School Sports Day Registration System

A web application designed to streamline student registration and event sign-ups for school sport days. Built with a **Flask** backend and a custom **HTML/CSS/JavaScript** frontend.

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)

---

## ✨ Features

* **Student Registration:** Form for students to submit details (Name, Grade/House, Contact Info).
* **Event Selection:** Dynamic selection of track and field events.
* **Form Validation:** Client-side JavaScript validation for clean data entry prior to submission.
* **Backend Processing:** Flask routes to handle form posts and store participant records safely.
